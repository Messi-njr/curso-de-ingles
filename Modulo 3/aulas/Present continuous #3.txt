Spealling changes = Mudanças ortográficas

Most verbs   add -ing = A maioria dos verbos adiciona -ing
Eat --------- Eating  = Coma --------- Comer
Work -------- Working = Trabalho -------- Trabalho

Verbs ending in -e   drop -e = Verbos terminados em -e eliminam -e

Take --------------- Taking = Pegue --------------- Tomando

Verbs ending in -ie   drop -ie   add -y  = Verbos terminados em -ie drop -ie add -y
Lie ------------------ Lying = Mentira ------------------ Mentira
Tie ------------------ Tying = Gravata ------------------ Amarrando

Verbs ending in c-v-c    double last letter = Verbos terminados em c-v-c com última letra dupla

Sit  ------------------- Sitting = Sente-se ------------------- Sentado
